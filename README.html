Text input

Step 1:

User must input typeface , size first ,and then the content.(If you don’t press “size” and “style” button , that’s ok.)

Step 2:

Press “TEXT” and then you can move cursor to the canvas, where you want to put the text, and click , the text will show on canvas successfully.

Note:

If you want to change size but don’t input typeface, it don’t work. You must input all elements.
Use these two text style to test this function:
Gloria Hallelujah/Permanent Marker
or you can add another href of font style you like.

Image input:

Step 1:

Add image link to the input block and press “IMAGE” button.

Step 2:

Then you can move cursor to the canvas, where you want to put the image, and click , the image will show on canvas successfully.

Note:

Please select smaller size of picture since my canvas can’t change size automatically. Image tool is the only function that can’t be downloaded.
